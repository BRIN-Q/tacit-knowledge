Codes for T1 measurements

Measurement codes for T1 Spin-Lattice Relaxation in NV Centers in Diamond.

Hardware : 
- NF Wavefactory 1973 AWG
- R&S SMC100A RF Generator
- Laser Obis LS 20 mW 532 nm
- G&H 532 nm  Fiber-Q AOM and Driver
- R&S RTM3000 Oscilloscope
- Thorlabs APD and Si Photodetector
- Optical setup
